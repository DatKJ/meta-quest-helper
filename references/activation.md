# Activation

用于回答 Quest 新机、恢复出厂后重新设置、手机配对和 Meta Horizon App 登录相关问题。

## Before Starting

先确认：

- 机型：Quest 2、Quest 3、Quest 3S、Quest Pro。
- 设备状态：全新、二手、已恢复出厂、是否还绑定旧账号。
- 手机：iOS 或 Android，是否已安装当前版本 Meta Horizon app。
- 账号：是否能登录 Meta 账号，是否完成邮箱验证和必要的安全验证。
- 网络：手机和头显是否连接稳定 Wi-Fi，路由器是否开启 AP 隔离，系统时间是否正确。
- 报错：卡在哪一步，是否有错误码、配对码、更新百分比或 App 提示。

## Official Activation Flow

1. 注册或登录 Meta 账号。优先使用本人长期可用的邮箱，按真实情况填写生日和安全信息。
2. 在手机安装并打开 Meta Horizon app，登录同一个 Meta 账号。
3. 打开 Quest，按头显提示选择语言、连接 Wi-Fi、等待必要更新。
4. 头显显示配对码后，在 Meta Horizon app 中添加设备并输入配对码。
5. 配对完成后，按提示完成边界、手柄、隐私、更新和基础设置。
6. 进入主界面后，检查系统更新、控制器固件、存储空间和账号状态。

## Network And Pairing Checks

如果设置卡住，按顺序排查：

1. 手机和 Quest 尽量连接同一个家庭 Wi-Fi。
2. 关闭路由器的 AP 隔离或访客网络隔离。
3. 确认手机蓝牙、定位、本地网络权限已授权给 Meta Horizon app。
4. 确认手机系统时间、Quest 系统时间和路由器时间正常。
5. 重启手机、Quest 和路由器后重试。
6. 换一条更稳定的家庭网络或移动数据环境做对比，但必须遵守所在地法律法规、网络管理要求和服务条款。
7. 如果仍失败，记录错误码和截图，再查看 Meta 官方帮助中心或联系官方支持。

不要给用户提供违反法律法规、平台规则、网络管理要求或服务授权的连接教程。

## Official Update Failure Handling

适合：新机首次更新失败、恢复出厂后更新卡住、更新后只显示 Meta logo、更新后无法进入系统。

先做：

1. 记录机型、系统提示、错误码、更新时间点和电量。
2. 保持头显接入可靠电源，不要在更新中途频繁断电。
3. 长按电源键完整关机，再开机确认是否能恢复。
4. 进入官方启动/恢复界面时，只选择 Meta 官方文档支持的选项。
5. 使用 Meta Software Update Tool 或 Meta 官方支持页面前，先核实当前官方说明。
6. 如果黑屏、无 UI、无法完成启动，优先联系 Meta 官方支持或购买渠道售后。

不要提供刷机、改系统分区、写入未知镜像、跳过官方校验或其他高风险系统修复流程。

## Used Device Checklist

二手机或他人使用过的设备先检查：

- 原主人是否已从设备退出账号并恢复出厂。
- 头显是否有 PIN、企业管理、家长控制、异常绑定或设备限制。
- 镜片是否划伤，屏幕是否有亮点、暗斑、烧屏，手柄是否漂移。
- Wi-Fi、蓝牙、追踪、边界、音频、麦克风、充电和 USB 数据连接是否正常。

## Referral, Bundle, And Region Notes

推荐奖励、套装游戏、商店活动、地区资格和到账时间经常变化。回答这类问题时先说明这是时效信息，需要以 Meta 当前官方页面、购买渠道页面和用户账号实际提示为准。

不要建议用户通过第三方账号、异常支付资料、虚假身份或违反规则的方式处理奖励、地区和购买问题。
