# 激活与首次配对

## 总原则

首次激活的关键不是“头显单独能上网”，而是手机上的 Meta Horizon App、Quest 头显、Meta 账号服务和同一局域网发现/蓝牙配对都能稳定工作。中国大陆用户常见失败点是：Meta 服务不可达、手机和头显不在同一可达网络、App 登录失败、账号地区/验证异常、二手机未恢复干净。

优先询问：

- 机型：Quest 2、Quest 3、Quest 3S、Quest Pro。
- 状态：全新未激活、恢复出厂后、二手机、已激活但换手机。
- 手机：iOS/Android，Meta Horizon App 是否能登录。
- 网络：家宽路由器、手机流量、电脑热点、路由器代理、境外/异地网络。
- 账号：是否能网页登录 Meta 账号，是否有双重验证、生日、地区、支付信息。

## 从零开始激活总流程

这个流程适合 Quest 3/3S 新机或恢复出厂后的设备。先做账号和奖励，再处理头显联网更新，最后手机 App 配对。

### 第一步：注册 Meta 账号

1. 打开 `meta.com` 注册或登录 Meta 账号，进入meta官网需要科学上网。
2. 优先使用邮箱注册，不建议用 Facebook 或 Instagram 直接关联注册，降低关联账号异常带来的连锁风险。
3. 邮箱优先使用稳定可收验证码的 Outlook、Gmail 或长期自用邮箱。
4. 姓名可用英文名或拼音，生日应按本人真实信息填写；未满 18 岁的账号可能受到应用、社交、开发者模式和购买限制。
5. 如果注册后收到封号或验证邮件，只在 Meta 官网重新登录并按官方提示做人机验证或身份验证；不要把账号、验证码或真人照片交给代激活人员。

### 第二步：领取推荐奖励

如果用户想领取 Meta 推荐奖励，必须在激活设备之前处理：

1. 先设置昵称和用户名；用户名建议使用英文和数字组合，后续使用 Virtual Desktop、好友搜索或社区协助时更容易识别。
2. 打开推荐邀请链接并确认已经接受邀请，再开始头显激活。
3. 推荐奖励金额、地区限制、购买来源、账号资格和到账时间会变化；`$30` 只是常见经验值，回答时必须提示激活前核实 Meta 当前规则。
4. 如果邀请链接报错，常见原因是账号地区、网络、购买来源或资格限制；不要建议购买黑产账号或代领服务。

### 第三步：联网和设备更新

1. 先选择下面“无法直连外网时的激活方案”，从方案 1 开始尝试。
2. 头显更新时保持充电，避免频繁断电或反复恢复出厂。
3. 如果更新卡在 0%，先重启头显和网络设备，确认 Quest 真的走了可达出口；仍失败时再临时关闭路由器 IPv6 做对比。
4. 如果使用手动 Wi-Fi 参数，IP、默认网关、DNS、代理地址必须逐字核对；Quest Wi-Fi 隐私/MAC 设置优先选择“使用设备 MAC”，避免路由器或加速器识别不到设备。

### 第四步：手机 App 配对

1. 手机安装 Meta Horizon App。iOS 用户通常需要能下载该 App 的 Apple ID 地区；Android 用户优先使用官方应用商店或可信来源，并核验安装包来源。
2. 打开 App 前，确认手机本身也能访问 Meta 服务；只让 Quest 可达而手机 App 不可达，配对仍会失败。
3. 在 App 中登录同一个 Meta 账号，输入头显显示的 5 位配对码。
4. 配对完成后，如果用户关心推荐奖励，在 App 的钱包/余额/商店相关页面确认是否到账；入口和命名以当前 App 为准。

### 第五步：开发者模式

开发者模式用于安装自有、开源或已授权 APK，不是激活必须步骤。激活完成后，如果用户需要 SideQuest、Quest 助手、未知来源、ADB 调试或安装合法 APK，统一读 `usage-tips.md` 了解用途和完整开启步骤；具体 ADB 命令再读 `adb.md`。

## 标准激活流程

1. 手机安装并登录 Meta Horizon App。
2. Quest 开机，连接 Wi-Fi，按头显提示更新或进入配对码界面。
3. 手机开启蓝牙、定位权限、局域网权限，和 Quest 连接到同一 Wi-Fi。
4. 在 App 中添加设备，输入或识别头显配对码。
5. 完成手柄、边界、安全教学和系统更新。

如果 App 登录、头显更新或配对阶段卡住，转入下面的大陆网络方案。

## 无法直连外网时的激活方案

按从简单到复杂推荐。先确认用户是否重视 Meta 推荐奖励、是否已有可用手机网络、是否能改路由器、是否有 Windows 电脑和基础网络知识。涉及“魔法”、代理、路由器分流时，提醒用户遵守所在地法律法规、网络管理规定和 Meta 平台条款，不推荐具体机场、节点商、代激活账号或黑产账号。

### 电脑端准备科学上网环境

适合：用户准备用电脑辅助激活、配置 SSTap、开 Windows 热点，或需要先让电脑能稳定访问 Meta 服务。

先解释“电脑端代理客户端”：这里指能在电脑上连接网络节点，并在本机提供 HTTP、SOCKS 或混合代理端口的软件。常见本机地址可能是 `127.0.0.1`，端口号以用户自己的客户端界面显示为准。回答时不要推荐或背书具体客户端、节点商、机场、下载站或代激活服务。

基础步骤：

1. 从可信来源下载电脑端代理客户端，安装前核验来源、版本、签名、项目维护状态和安全风险。
2. 购买或获取合规节点/订阅；优先确认稳定性、退款规则、UDP 支持、设备数限制、隐私政策、日志政策和账号风险。
3. 在客户端中导入订阅链接，或按服务提供方给出的信息手动填写节点。
4. 更新订阅并选择节点，确认客户端已经连接。
5. 用电脑浏览器测试 `meta.com`、Meta 账号登录页、Meta Horizon 相关页面是否能正常打开。
6. 记录本机代理地址、端口号和代理类型，例如 `127.0.0.1`、HTTP/SOCKS/混合端口；后续给 SSTap、Windows 热点共享、手动代理参数或排障时会用到。

注意：

- 订阅链接、节点信息、支付信息和 Meta 账号验证码都不要交给陌生人。
- 如果后续要走 SSTap 方案，节点必须支持 UDP；只支持网页浏览的节点不一定能完成 Quest 激活、更新或多人服务。
- 不要承诺任何节点、客户端或线路一定可用；平台规则、网络环境和服务质量都会变化。

### 方案 1：UU 加速器

适合：想最快完成首次激活、没有路由器配置能力、希望按图形界面操作的新手。

准备：

- 能登录 Meta Horizon App 的手机。
- 支持 Quest/Meta 商店加速的 UU 加速器客户端和账号。
- 若使用电脑模式，准备一台与 Quest 在同一局域网的 Windows 电脑。
- 若使用路由器模式，确认路由器后台支持 UU 加速器或相关插件。
- Quest 电量至少 50%。

步骤：

1. 先在手机上确认 Meta Horizon App 能打开、登录、加载账号头像/设备页。
2. 优先尝试 UU 路由器模式：如果路由器支持 UU，在路由器后台开启 Quest/Meta 相关加速，让手机和 Quest 连接同一 Wi-Fi。
3. 如果没有路由器模式，用电脑模式：电脑下载并打开 UU 加速器，搜索 `Meta 商店`、`Meta Quest` 或当前客户端提供的 Quest 相关加速项并开启。
4. 按 UU 当前界面提示，把显示的 IP、默认网关、DNS、代理等参数逐字填入 Quest 的 Wi-Fi 手动设置。数字不要猜，填错一个段都会导致更新失败。
5. Quest Wi-Fi 隐私/MAC 选项优先选择“使用设备 MAC”，避免加速器或路由器因随机 MAC 识别不到头显。
6. 确保 Quest、电脑/路由器、手机 App 在同一可用网络环境下，再在 Meta Horizon App 中添加设备。
7. 等待系统更新和配对完成；激活完成后，再根据长期使用需求选择手机热点、路由器 Wi-Fi 或 PCVR 网络方案。

注意：

- UU 加速器流程可能影响或无法领取 Meta `$30` 推荐奖励。若用户重视推荐奖励，必须在激活前核实推荐链接、账号地区、购买来源和 Meta 当前奖励规则。
- UU 客户端入口、支持机型和流程会变化；回答“现在怎么点”时需要让用户按当前客户端页面核实。
- 如果更新卡在 0%，先重启 Quest、UU 加速器、路由器或电脑网络；仍失败时检查手动参数、关闭路由器 IPv6 做对比，最后再考虑恢复出厂。
- 恢复出厂会清除本机数据；新机激活阶段影响较小，但仍要提醒用户确认没有重要截图、录屏或离线数据。

### 方案 2：手机有魔法开热点给 Quest 连接

适合：手机已经有可达 Meta 服务的网络，想临时激活，不想改路由器。

准备：

- 手机端能稳定登录 Meta Horizon App。
- 手机热点可开启，SSID 使用英文和数字。
- Quest 电量至少 50%。

步骤：

1. 手机先打开魔法/可达网络，并确认 Meta Horizon App 能正常登录。
2. 手机开启热点，让 Quest 连接该热点。
3. Quest 进入配对码界面后，在手机 App 中添加设备。
4. 完成配对、基础设置和必要系统更新。
5. 激活完成后再切换到长期网络方案。

失败排查：

- 部分手机无法把 VPN/代理共享给热点，表现为手机能打开 Meta，Quest 不能更新或商店空白。此时换方案 3 或方案 4。
- App 能登录但找不到头显：确认蓝牙、定位、局域网权限；手机与 Quest 靠近；重启 App 和 Quest。
- 热点不稳定：改 5 GHz/2.4 GHz，关闭省电模式，SSID 不使用中文或特殊字符。

注意：

- 此方法失败率较高。

### 方案 3：路由器 Wi-Fi 有魔法

适合：想长期稳定使用 Quest，家里有可配置路由器、软路由或旁路由。

准备：

- 路由器或旁路由已配置可达 Meta 服务的出口。
- 手机和 Quest 连接同一个 SSID。
- 2.4 GHz/5 GHz 分开命名，优先让 Quest 使用 5 GHz。

步骤：

1. 在路由器上确认 Meta 账号、系统更新、商店、下载相关流量都走可达出口。
2. 给手机和 Quest 连接同一个 SSID，避免手机走蜂窝数据而头显走 Wi-Fi。
3. 关闭访客网络、AP 隔离、客户端隔离、过度严格的广告拦截和 DNS 劫持。
4. 手机 App 登录成功后添加 Quest。
5. 激活完成后保留此网络用于商店、下载、系统更新和多人联机。

失败排查：

- App 可以登录但头显不行：检查路由规则是否只覆盖手机或浏览器，没有覆盖 Quest。
- 头显下载慢或失败：换 5 GHz、靠近路由器、临时关闭 IPv6 或调整 DNS 做对比。
- 配对发现失败：确保手机和 Quest 在同一网段，没有 AP 隔离；手机不要连接访客 Wi-Fi。

### 方案 4：SSTap 全局 UDP 热点终极方案

适合：这是四个方案里配置最复杂的终极方案。适用于方案 1-3 都失败、不能改路由器、手机热点不可用，但有 Windows 电脑、无线网卡和基础网络操作能力的用户。只要节点支持 UDP、SSTap 测试通过、Windows 共享适配器选对、热点稳定，成功率通常较高。

如果用户不理解电脑端代理客户端、节点订阅、本机地址或端口号，先按上面的“电脑端准备科学上网环境”解释清楚，再进入 SSTap 设置。

准备：

- Windows 电脑。
- SSTap。
- 已可用的电脑端代理客户端；如果直接在 SSTap 中导入或使用服务器节点，则不一定需要额外代理客户端。
- 无线网卡：笔记本通常自带无线网卡；台式机通常需要额外插入 USB 或 PCIe 无线网卡，才能开启 Windows 移动热点。
- Quest 电量至少 50%，并准备好 Meta Horizon App 配对。

配置节点和 UDP 测试：

1. 打开 SSTap，添加可用节点。
2. 如果电脑端代理客户端已经可用，可在 SSTap 中填写本机代理地址，常见为 `127.0.0.1`，端口号按客户端实际显示填写。
3. 如果用户有可被 SSTap 直接使用的服务器节点，也可以在 SSTap 内直接导入或配置节点。
4. 在 SSTap 中点击测试，重点看 UDP 转发测试结果。
5. UDP 转发必须通过。UDP 不通时，Quest 激活、系统更新、多人联机或部分 Meta 服务可能失败，不建议继续往下做。

开启 Windows 移动热点：

1. 打开 Windows 设置。
2. 进入“网络和 Internet”。
3. 打开“移动热点”。
4. 记录热点名称和密码。
5. 如果热点无法开启，检查无线网卡驱动、网卡是否被禁用、台式机是否确实有无线网卡。

共享 SSTap 网络到热点：

1. 进入 Windows 的“高级网络设置”或传统“网络连接”页面。
2. 找到名为 `SSTap` 的网络适配器。
3. 打开该适配器属性，进入共享选项。
4. 勾选“允许其他网络用户通过此计算机的 Internet 连接来连接”。
5. 在“家庭网络连接”下拉菜单中，选择 Windows 移动热点对应的“本地连接”或“本地连接* 数字”。这个选项必须选对，选错会导致 Quest 能连上热点但没有可用网络。
6. 保存设置后，回到 SSTap 主界面点击连接。

激活 Quest：

1. 戴上 Quest，连接电脑刚开启的移动热点 Wi-Fi。
2. 手机 Meta Horizon App 登录同一个 Meta 账号，并确认手机端也能访问 Meta 服务。必要时手机也连接同一个电脑热点，或让手机自行使用可达网络。
3. 在 App 中添加设备，输入头显里的 5 位配对码。
4. 等待 Quest 自动激活和更新。若下载速度能达到约 `2-3 MB/s`，首次激活和更新可能十几分钟完成；实际时间取决于系统版本、服务器、节点和 Wi-Fi 稳定性。
5. 激活完成后，头显内商店能正常显示内容，可作为当前网络配置成功的参考信号。

失败排查：

- Quest 能连热点但无网：优先检查共享时“家庭网络连接”是否选中了移动热点对应的“本地连接/本地连接* 数字”。
- 更新卡在 0%：重新测试 SSTap 的 UDP 转发；确认 SSTap 已连接；确认节点支持 UDP；必要时换节点或换回方案 1-3。
- Quest 找不到电脑热点：检查无线网卡、热点是否开启、SSID 是否包含中文或特殊字符、2.4 GHz/5 GHz 兼容性。
- 手机 App 能登录但 Quest 不行：确认 Quest 连接的是电脑热点，确认热点流量确实经过 SSTap，而不是只代理了电脑浏览器。
- 配对失败：手机和 Quest 需要能完成蓝牙、局域网发现和 Meta 账号服务访问；不要让手机和 Quest 处在完全隔离的网络里。
- Windows 防火墙、网卡优先级、DNS、代理模式、共享适配器选错都可能导致失败。
- SSTap、电脑端代理客户端、节点来源和系统兼容性需用户自行核实安全性、稳定性和合法性。

风险提示：此方案是成功率较高但配置成本也最高的终极方案。回答时先提醒用户遵守所在地法律法规、网络管理规定和 Meta 平台条款，不推荐具体节点商、机场、绕过监管服务、代激活账号或黑产账号。


## 常见激活失败

- App 登录失败：先解决手机到 Meta 服务的可达性，再尝试配对。
- 卡在更新：保持充电，换网络，等待 30 分钟以上；不要频繁断电。
- 找不到头显：手机和 Quest 同网，开启蓝牙/定位/局域网权限，关闭 AP 隔离。
- 账号验证卡住：用浏览器登录 Meta 账号，完成邮箱、手机、2FA 或生日验证。
- App 商店地区异常：不要频繁切换地区或支付资料；购买前先确认账号地区与支付方式风险。
- 推荐奖励未到账：确认是否在激活前接受邀请、账号地区和购买来源是否满足 Meta 当前规则；不要通过代领、共享账号或异常支付资料处理。
- 附赠游戏无法领取：常见原因是账号地区、购买渠道、活动资格或时间限制；例如某些套装游戏可能仅限特定地区账号，回答前需要联网核实当前规则。

## 官方信息核实

涉及 Meta Horizon App 名称、设备设置流程、账号政策和开发者模式入口时，以 Meta 官方帮助中心和 Meta Horizon 开发者文档为准。若用户问“现在最新流程”，先联网核实。
