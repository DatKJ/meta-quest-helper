# Content Recommendations

用于按用户兴趣推荐 Quest 内容类型和体验顺序。具体 app、游戏、价格、地区、套装和平台支持属于时效信息，回答前应核实当前可用性。

## Ask First

- 用户是否第一次玩 VR。
- 是否容易晕动。
- 喜欢健身、音乐、射击、解谜、社交、观影、创作还是学习。
- 是否有足够活动空间。
- 是否需要单人、多人、本地同屏观看或家庭共享。

## Beginner-Friendly Categories

- 固定站位节奏类：适合入门，移动少，容易建立 VR 适应。
- 桌面模拟类：如乒乓、棋类、桌游类，舒适度通常较高。
- 绘画和创作类：适合不追求强刺激、喜欢慢节奏的人。
- 观影和空间视频类：适合熟悉头显界面和佩戴方式。
- MR 体验类：能看到现实环境，新手安全感更强。

## Fitness Categories

推荐思路：

- 音乐节奏：适合持续有氧、情绪释放。
- 球拍运动：适合有真实运动经验的人。
- 拳击训练：强度高，先确认空间和身体状态。
- 家庭运动：适合轻量活动和多人轮流体验。

运动提醒：

- 系腕带。
- 清空活动空间。
- 不适或心率过高时立即停止。
- 建议活动空间不小于 `2 x 2 m`。
- 初次体验不要连续玩太久。

## WebXR Guidance

WebXR 是浏览器内的沉浸式网页体验。推荐时不要只丢链接，先说明适合用来做什么：

- 浏览器兼容性测试。
- 简单 3D 展示。
- 教学演示。
- 轻量互动体验。

可用性会随浏览器版本、设备系统和页面维护状态变化，回答前需要现场核实。

## Content Trend Categories

以下内容兴趣点可以作为推荐入口，但不要把具体 app、移植项目、试玩资格、价格或可用性说死。

- 2D PC 游戏大屏：适合想把 Quest 当私人巨幕的人，重点看输入焦点、延迟、字体大小和长时间佩戴。
- 复古游戏：适合怀旧和轻量体验，先确认来源、授权和控制器适配。
- 3D 电影和空间媒体：适合观影用户，重点问片源格式、字幕、存储空间、舒适头带和电池。
- MR 房间扫描和轻量 MR 小游戏：适合展示 Quest 的混合现实能力，提醒用户清理空间和保护隐私。
- 独立开发者试玩：适合愿意反馈 bug 的玩家，提醒用户核实项目来源、测试资格、隐私权限和退出方式。
- 大作移植或独立项目：先说明支持平台、控制方式、版权和安装方式需要核实，不提供不合规来源。

## Recommendation Style

回答时按用户画像给 3-5 个方向即可：

```text
你是新手且担心晕动：先从固定站位、节奏类、桌面模拟类开始。
你想健身：优先选择节奏有氧和球拍运动，拳击类强度更高。
你想观影：优先考虑舒适头带、镜片清洁、长时间佩戴和电池方案。
```
