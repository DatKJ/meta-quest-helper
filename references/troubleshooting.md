# Troubleshooting

用于通用排障。先收集现象、环境、最近变化，再给最短路径。

## Info To Collect

- 机型和系统版本。
- 新机、二手、恢复出厂后还是日常使用。
- 手机系统和 Meta Horizon app 版本。
- 报错文字、错误码、截图。
- 最近是否系统更新、换路由器、换账号、安装新应用或摔碰设备。

## Pairing Fails

可能原因：蓝牙权限、本地网络权限、AP 隔离、配对码过期、账号不一致。

先做：

1. 手机和 Quest 重启。
2. 确认手机蓝牙、定位、本地网络权限开启。
3. 手机和 Quest 连接同一家庭 Wi-Fi。
4. 关闭访客网络和 AP 隔离。
5. 重新生成配对码。

## Update Stuck

先做：

1. 保持充电。
2. 等待一段时间，不要频繁断电。
3. 重启头显和路由器。
4. 确认存储空间足够。
5. 换稳定网络环境做对比。

仍失败时，记录错误码并查询 Meta 官方帮助中心。

## Black Screen After Update

典型现象：更新后只显示 Meta logo、随后黑屏、没有 UI 或启动声音。

先做：

1. 充电 30 分钟以上，确认不是低电量。
2. 长按电源键完整关机，再开机。
3. 进入官方启动/恢复界面时，只使用 Meta 官方说明支持的选项。
4. 尝试 Meta Software Update Tool 前先核实当前官方帮助页面。
5. 如果恢复出厂后仍黑屏，优先联系 Meta 官方支持或购买渠道售后。

不要建议用户写入未知系统文件、改系统分区或执行来源不明的修复脚本。

## Charging Stuck

适合：充电卡在固定百分比、橙灯常亮或闪烁、长时间不用后电量异常。

1. 换原装或可靠充电器和线材，避免低功率 USB 口。
2. 关机充电 1-2 小时后再开机查看。
3. 检查 USB-C 口是否有灰尘、松动、进液或烧蚀痕迹。
4. 移除第三方电池头带或转接线，直接给头显充电。
5. 如果多次卡在同一百分比，记录充电器功率、线材、LED 状态和系统版本。
6. 仍异常时优先走保修或售后，不建议自行拆机。

## Controller Pairing

1. 更换电池。
2. 清洁电池仓触点。
3. 在 Meta Horizon app 中重新配对手柄。
4. 确认房间光线充足，没有大面积反光。
5. 检查手柄是否摔坏或摇杆漂移。

## Tracking Lost

1. 打开房间灯光，避免太暗或强反光。
2. 擦拭外部追踪摄像头。
3. 清除并重新设置边界。
4. 避免镜子、玻璃、纯白墙面和快速闪烁灯光。
5. 重启头显。

## Passthrough Flashes During Gameplay

适合：游戏中突然闪出 Passthrough 一瞬间、画面被打断、游戏暂停或方向被转动。

可能原因：边界误判、双击透视触发、光线变化、镜面反光、传感器污渍、系统版本问题。

先做：

1. 清洁外部摄像头。
2. 重新绘制边界并清理旧边界数据。
3. 遮挡或移开镜子、玻璃和强反光物。
4. 检查是否误触双击透视或手势快捷操作。
5. 换一个光线稳定的房间测试。
6. 如果更新后才出现，记录系统版本和复现条件，优先查看官方支持信息。

## Gesture Or Body Tracking Glitches

适合：Pinch 手势失效、任务栏拖不动、手臂长度识别异常、手部追踪和手柄射线表现不一致。

1. 先切换手柄操作，确认不是应用本身卡死。
2. 检查手部追踪开关、房间光线和外部摄像头清洁。
3. 重启头显后再测试同一动作。
4. 重新设置边界和身体/地面高度。
5. 移开镜面、玻璃桌、强反光地面和过暗背景。
6. 如果恢复出厂也无效，考虑系统版本问题或硬件问题，保留视频证据并联系官方支持。

## Proximity Sensor Or Random Black Screen

适合：戴上后黑屏、摘下又亮、清洁接近传感器后短暂恢复、反复黑屏。

1. 清洁眉心附近的接近传感器区域。
2. 确认面罩、眼镜架、第三方配件没有遮挡传感器。
3. 移除临时贴纸、遮挡物和不合适的面罩做对比。
4. 重启头显并测试不同佩戴位置。
5. 如果清洁和配件排除后仍反复黑屏，优先保修或售后。

## App Download Or Update Stuck

适合：应用下载卡在固定大小、显示 `0 MB / x GB`、更新无限加载。

1. 确认剩余存储空间足够，至少预留大于安装包数倍的空间。
2. 暂停下载后重启头显再继续。
3. 取消任务后从应用库或商店重新开始。
4. 检查系统时间和账号状态。
5. 换稳定 Wi-Fi 做对比。
6. 仍失败时记录 app 名称、版本、卡住大小和截图，查询 Meta 官方支持。

## Random Background Music After Standby

适合：待机一两天后唤醒，出现无法关闭的背景音乐或环境音。

1. 打开最近应用列表，逐个关闭最近运行的应用。
2. 检查浏览器、商店、视频应用和系统空间是否仍在播放音频。
3. 切换音量输出，确认不是蓝牙设备或投屏残留。
4. 重启头显。
5. 如果每次待机后复现，记录系统版本和最近打开的应用。

## Video Quality Drops

适合：视频 app 或网页播放几分钟后从高清变糊，但其他设备正常。

1. 先确认同一 Wi-Fi 下手机或电脑播放是否正常。
2. 在视频服务账号中检查播放质量设置。
3. 关闭后台下载和投屏。
4. 重启浏览器或视频 app，必要时清理缓存。
5. 换 5 GHz Wi-Fi 或靠近路由器测试。
6. 如果只有某个视频服务异常，优先按该服务的 Quest/浏览器兼容性处理。

## Blurry Image

1. 调整 IPD。
2. 重新佩戴，让甜区对准眼睛。
3. 用超细纤维布轻擦镜片。
4. 戴眼镜用户确认镜片不互相接触。
5. 检查是否需要处方镜片。

## Motion Sickness

1. 立即停止，摘下头显休息。
2. 下次从 15-30 分钟开始。
3. 优先选择固定站位、瞬移移动、低强度体验。
4. 关闭平滑转向，使用分段转向。
5. 不舒服时不要硬撑。

## High-Risk Beginner Mistakes

- 不要让阳光直射目镜镜片。
- 戴眼镜时确认眼镜镜片不接触 VR 镜片。
- 不要用刷子刷洗织物面罩。
- 运动空间不要小于 `2 x 2 m`，并系好腕带。
