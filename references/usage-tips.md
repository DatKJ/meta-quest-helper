# Usage Tips

用于回答日常使用技巧：开发者模式、未知来源、中文语音输入法、授权应用安装、2D 应用设置入口、清洁和安全习惯。

## Developer Mode

开发者模式适合：

- 调试自己开发的 Quest/Android 应用。
- 安装本人拥有授权的测试 APK。
- 使用 ADB 读取日志、截图、录屏或传输个人文件。
- 排查 Unknown Sources/未知来源不显示的问题。

不要把开发者模式描述成规避付费、绕开授权或获取不当内容的工具。

### How To Enable Developer Mode

1. 使用本人 Meta 账号登录 Meta Horizon app。
2. 登录 Meta 开发者相关页面，创建或加入一个开发者组织。
3. 按官方要求完成账号或组织验证。验证方式以当前官方页面为准。
4. 确认头显已在 Meta Horizon app 中完成配对。
5. 在 Meta Horizon app 中进入设备设置，找到 Developer Mode 并打开。
6. 重启 Quest。
7. 第一次用 USB 连接电脑时，在头显内确认 USB debugging 授权。

常见失败：

- App 中看不到 Developer Mode：确认开发者组织和验证是否完成，退出重登 App。
- 开关打开后仍无效：重启头显，检查是否使用头显主账号。
- 电脑识别不到设备：换数据线、换 USB 口、重新确认头显内授权弹窗。

## Unknown Sources Not Showing

按顺序排查：

1. 使用头显主账号操作。
2. 确认 Developer Mode 已开启并重启过头显。
3. 安装一个本人拥有授权的测试 APK。
4. 在应用库中切换筛选器，查找 Unknown Sources/未知来源。
5. 更新 Meta Horizon app 和头显系统。
6. 重新插拔 USB 并确认 USB debugging 授权。
7. 多账号环境下先用主账号测试。
8. 如果仍失败，收集系统版本、账号状态、APK 安装报错和 `adb devices` 结果。

## Chinese Voice Input Method

适合希望在 Quest 的 2D 应用中更方便输入中文的用户。输入法会接触文本、语音和剪贴板内容，务必使用可信来源。

流程：

1. 在电脑准备与 Android 兼容的中文语音输入法 APK，优先选择官方来源或可信开发者来源。
2. 用官方开发者工具、可信设备管理工具或 `adb install` 将 APK 安装到 Quest。
3. 如需进入系统 2D 设置，可安装可信的 Quest 设备管理工具，并从 2D 应用列表打开系统设置入口。
4. 在系统输入法设置中关闭不需要的默认输入法，开启刚安装的中文语音输入法。
5. 回到需要输入的应用中测试中文键盘、语音输入和候选词。

注意：

- 不安装来源不明或二次打包的 APK。
- 不把密码、验证码、支付信息输入到不可信输入法。
- 系统版本更新后，入口名称或设置路径可能变化，需要按当前界面调整。

## Daily Safety Tips

- 不要让阳光直射 Quest 镜片。镜片会聚光，可能造成永久显示损伤。
- 戴眼镜使用时确认眼镜镜片不接触 VR 镜片。最好使用镜片垫片或处方镜片。
- 不要用刷子刷洗织物面罩。用温水轻按清洗后阴干。
- 运动类内容建议至少预留 `2 x 2 m` 活动空间，佩戴腕带，移走桌角、玻璃杯和宠物用品。
- 第一次体验控制在 15-30 分钟，不适时立刻停止。

## Productivity And Window Tips

很多用户想把 Quest 当虚拟桌面、移动工作站或写代码屏幕。回答时不要只看“能不能开多窗口”，还要确认：

- 单次工作时长，是 20 分钟体验还是 2 小时以上办公。
- 是否需要读代码、表格、CAD 小字或网页正文。
- 当前头带是否压脸，是否有后置配重或顶部支撑。
- 是否容易眼干、头痛、颈部疲劳或读字疲劳。
- 是否需要看键盘、鼠标、手机和现实桌面。

建议：

1. 先试 20-30 分钟真实工作任务，不要只看演示画面。
2. 读字模糊时先调 IPD、佩戴高度、窗口距离、字体大小和系统缩放。
3. 长时间办公优先改善头带、面罩、散热和电池方案。
4. 每 30-45 分钟摘下休息一次。
5. 如果目标是全天写代码或 CAD，提醒用户 Quest 更适合短时虚拟桌面，不一定适合替代传统显示器。

## Gesture And Multi-Window Glitches

适合：Pinch 手势失效、任务栏拖不动、多窗口无法整体移动、手柄射线正常但手势无效。

先做：

1. 切换到手柄操作，确认只是手势问题还是整个 UI 卡住。
2. 重启头显，不要一上来恢复出厂。
3. 检查手部追踪是否开启，房间光线是否足够。
4. 清洁外部追踪摄像头，移开镜面和强反光物。
5. 检查系统更新说明，判断是否可能是版本问题。
6. 如果更新后才出现，记录系统版本和复现视频，优先走官方支持或等待修复。
